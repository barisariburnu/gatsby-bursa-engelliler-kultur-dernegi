# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial open-source release
- MIT License
- Contributing guidelines
- Comprehensive documentation
- Environment variable configuration examples
- Multi-template Gatsby blog support (9 templates)

### Changed
- Refactored hardcoded URLs to use environment variables
- Updated documentation for better clarity
- Improved project structure for open-source distribution

### Fixed
- Environment variable handling
- Configuration consistency across templates

## [2.2.0] - 2025-11-25

### Added
- Initial version with 9 Gatsby blog templates:
  - Agency Blog
  - Agency Blog Modern
  - Beauty Blog
  - Personal Blog
  - Personal Blog Minimal
  - Personal Blog Lite
  - Image Blog
  - Photography Blog Minimal
  - Romantic Blog (Tinder-like)

### Features
- TypeScript support
- Styled Components with Styled System
- Lerna monorepo structure
- Yarn workspaces
- Disqus comments integration
- Mailchimp newsletter integration
- Instagram feed integration
- Google Analytics support
- Markdown blog posts with frontmatter
- SEO optimization
- PWA support
- RSS feed generation
